

# 安装&升级

```
bash <(curl -Ls https://raw.githubusercontent.com/xxf185/x-ui/master/install.sh)
```

## 安装指定版本

要安装所需的版本，请将该版本添加到安装命令的末尾。例如:v2.3.6

```
bash <(curl -Ls https://raw.githubusercontent.com/xxf185/x-ui/master/install.sh) v2.3.6
```
